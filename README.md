# Full Stack Open - Course Progress

## Overview

This repository contains my completed exercises and projects from the **[Full Stack Open](https://fullstackopen.com/)** course by the University of Helsinki. The course provides a deep dive into modern JavaScript-based web development, covering React, Redux, Node.js, MongoDB, GraphQL, TypeScript, React Native, CI/CD, Containers, and Relational Databases.

**Technologies Learned:** React, Redux, Node.js, Express, MongoDB, GraphQL, TypeScript, React Native, Docker, PostgreSQL, CI/CD with GitHub Actions
